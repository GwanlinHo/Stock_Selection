# 台股「四層過濾網」AI 自動化選股系統

這是一套根據網路高手選股邏輯開發的自動化系統，旨在將全市場 2,000+ 檔標的，透過四層過濾網（技術、量能、籌碼、基本面），精煉出最後 5 檔優質觀察名單。

本系統採用 **Python (高併發計算)** 與 **Gemini CLI Agent (智能決策)** 的協作模式。

## 🚀 快速開始

### 1. 安裝環境
本專案使用 `uv` 進行管理。請確保您已安裝 `uv`，然後執行：
```bash
uv sync
```

### 2. 配置金鑰 (選用)
為了獲取法人數據，建議註冊 FinMind 並獲取 Token。
建立 `.env` 檔案：
```bash
FINMIND_TOKEN=您的_Token_在此
```

### 3. 一鍵啟動
在 Gemini CLI 中輸入以下指令：
> **`stock selection`**

系統會自動執行全市場掃描、增量數據更新、AI 新聞分析，並在 `reports/` 下產出詳細週報。

---

## 🛠️ 策略架構：四層過濾漏斗

1.  **L1 技術面過濾**：股價站在月線 (20MA) 之上，且月線、季線趨勢向上。
2.  **L2 成交量過濾**：篩選具備流動性的人氣股 (日均量 > 1000/5000 張)。
3.  **L3 籌碼面過濾**：串接 API 檢查外資或投信近一個月的累計買超動向。
4.  **L4 基本面與 AI 決策**：
    *   **硬性指標**：月營收年增率 (YoY) 為正，最新一季 EPS 為正。
    *   **軟性過濾**：由 AI Agent (Gemini) 自動搜尋新聞，排除利空並進行 A/B/C 分類。

---

## ⚙️ 標準管理 (Config)

您可以在 `config/settings.json` 中切換不同的篩選層級：
*   **Strict (嚴格)**：適用於高勝率中長線佈局。
*   **Neutral (中性)**：一般市場環境下的標準配置。
*   **Loose (寬鬆)**：用於盤整市或尋找落後補漲標的。

---

## 📊 報告產出
每次掃描會產出兩份 Markdown 文件：
*   **`reports/SCAN_DATA_YYYY-MM-DD.md`**：全市場過濾後的詳盡數據清單。
*   **`reports/WEEKLY_REPORT_YYYY-MM-DD.md`**：由 AI 撰寫的精選觀察名單，含數據面板與新聞摘要。

---

## 📁 目錄結構
*   `src/`: 核心計算原始碼 (價量計算、API 串接)。
*   `data/cache/`: 增量更新數據快取 (Parquet 格式)。
*   `reports/`: 選股報告存檔。
*   `config/`: 篩選參數配置。
*   `GEMINI.md`: AI Agent 的執行手冊。
